
🚀 Découvrez ma toute nouvelle application développée avec React Native ! 📱✨

Il s'agit d'une calculatrice offrant des operations de base pour des calculs mathetique simple.

J'ai utilisé le hook `useReducer` de React pour gérer efficacement toute la logique de l'application, offrant une expérience de calcul fluide et rapide. 🧮💨

Le design est inspiré d'un article sur Dribbble, apportant une esthétique moderne et conviviale à l'application. 🎨
